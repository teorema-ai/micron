# micron
Models for miRNA space learning.

# SETUP
* micron
    - local install: 
    ```
        git clone git@github.com:pearson-laboratory/micron.git $HOME/micron
        pip install -e $HOME/micron
    ```

# NOTEBOOKS
```
    notebooks/mircoseqs/mircoseqs_hnsc_fasttext.ipynb
```
